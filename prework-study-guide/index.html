HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Text is organized using numbered headings, with "h1" denoting the largest font of header text, "h2" next in size, and so on.
  • Each element of code is started with a tag (< >) and ended with a tag and backslash (< />)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS page must be linked to in the "head" section just after the listed metadata to be applied to the HTML.
  • Each CSS rule is began with a selector, defining the element to which the rules will apply, followed by the declarations in brackets, containing the property we want to apply and its value (e.g. "color: blue")

Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: switches to the specified branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "insert message here": -m denotes a message to the commit on Git Hub
  • git pull origin main: pulls the base branch to the local feature branch (to make sure it is up to date before pushing changes)
  • git push origin feature/branch-name: pushes local changes to the GitHub repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can control the order in which coe is executed.
  • If (& else if) statements allow code to only execute if the conditions of the statement are met.
  • An array is a single variable that is used to hold a group of data, usually related in some way.
  • Arrays are 0-indexed and sequential in JS (and many other programming languages)
  • For loops use the prdictable pattern of indeces to perform a task on all the items in an array by allowing a single code block to be executed over and over, interrupted by a conditional if statement.
  • For loop set-up: for(let x = 0; x < variable.length; x++) { //code block; }
  • A function is a set of instructions that tells the comnputer how to perform a certain task.
  • Functions do not automatically execute, they must be called by name in order to execute.
  • A function's name is ALWAYS followed by open and close parentheses ().
